# Live Avatar Agent - Python Backend

## Description
This project serves as the backend for the Live Avatar Agent developed in Python.

## Setup Instructions

### Prerequisites
- Python 3.x
- pip (Python package installer)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Uncle-Becky/live-avatar-agent.git
   cd live-avatar-agent
   ```
2. Create a virtual environment:
   ```bash
   python -m venv venv
   ```
3. Activate the virtual environment:
   - On Windows:
     ```bash
     venv\Scripts\activate
     ```
   - On macOS and Linux:
     ```bash
     source venv/bin/activate
     ```
4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Configuration
- Create a `.env` file in the root directory and add required environment variables:
  ```env
  ENV_VARIABLE_NAME=value
  ```

### Running the Application
- Start the backend server:
  ```bash
  python app.py
  ```
